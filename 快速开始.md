## 🚀 快速测试 3 步骤

- 生成 1 个测试卡密（有效期 30 天）
- 记录卡号

### 2️⃣ 安装应用到 iPad

**推荐方式：使用 Filza**

```bash
# 将 IPA 复制到 iPad
scp build/WechatVideoReplacer.ipa root@iPad的IP:/var/mobile/Documents/

# 在 iPad 上用 Filza 打开并安装
```

### 3️⃣ 启动测试

1. 启动应用
2. 输入测试卡号
3. 点击 "验证"
4. 成功后自动进入主界面
